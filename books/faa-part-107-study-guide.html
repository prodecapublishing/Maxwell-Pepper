Frequently Asked Questions

Is this book up to date with the current Part 107 rules? +
Yes — this guide is written to reflect current FAA Part 107 regulations. Maxwell Pepper Books reviews and updates guides when significant regulatory changes occur. Check the Amazon listing for the latest edition date.
Do I need any aviation knowledge to use this book? +
No prior aviation knowledge is required. The book is written for complete beginners to aviation concepts. All terminology is explained clearly, and complex topics like METARs and sectional charts are broken down step-by-step.
Does the book include practice questions? +
Yes — each chapter includes review questions to test your understanding. The book also recommends supplementary practice test resources to build exam readiness alongside the content.
Is there a Kindle version available? +
Yes. The Kindle version is available on Amazon and contains the same complete content as the paperback. Both formats are ideal for study — paperback for note-taking, Kindle for mobile reading on the go.
